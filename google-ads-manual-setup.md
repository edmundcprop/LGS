# Google Ads — Manual Setup Reference

Use this doc while building your 5 Search campaigns in the Google Ads UI. Each section below is one campaign. Open each campaign in Google Ads, then copy/paste from here.

**Setup order per campaign:**
1. Open the campaign
2. Pause/delete any auto-created "Ad group 1" placeholder
3. Create a new ad group with the name shown below
4. Add all keywords (paste the block — Google Ads accepts one keyword per line)
5. Create one Responsive Search Ad with the headlines, descriptions, and final URL shown below
6. Verify status: ad group enabled, keywords enabled, ad enabled, campaign enabled

## Ad Schedule (apply to ALL 5 campaigns)

Run daily from **09:00 to 18:00** Malaysia time, Monday through Sunday.

In the campaign settings → Ad schedule → add 7 entries (one per day) with start time 09:00 and end time 18:00.
